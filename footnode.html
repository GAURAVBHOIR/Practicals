The precise reader should note that it is not correct to refer to a confidence interval as giving the probability of the true parameter value being in a given range. Such a statement is not possible in classical statistics. What the confidence interval actually shows is the probability that a new observation would give a certain parameter value, assuming that the best-fit value obtained for the current observation is the true value of the parameter
